## Introduction (Snoozeloo)

This repository is dedicated to my ongoing journey in mastering key aspects of software development, particularly focusing on patterns and frameworks used in modern Mobile applications. The goal is to enhance my understanding and practical skills in the following areas:

• Clean Architecture: A robust structural approach that keeps business logic decoupled from frameworks.

• MVI/MVVM: Model-View-Intent or Model-View-ViewModel, which are patterns used for structuring apps and enhancing UI development efficiency.

• Offline-first App: Room library for local database in applications.

• Dependency Injection (DI): A technique to inject dependencies such as data repositories into classes rather than having each class create its own instance.

• Navigation Component: Helps structure and manage the navigation graph of an application, providing a unified approach for screen transitions.

• Jetpack Compose: Toolkit for building native Android UI with Kotlin and other JVM languages
